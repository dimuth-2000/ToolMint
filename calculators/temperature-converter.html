Temperature Converter

Convert temperature between Celsius, Fahrenheit, and Kelvin.

Formula

  • C → F: F = (C × 9/5) + 32
  • F → C: C = (F − 32) × 5/9
  • C → K: K = C + 273.15
  • K → C: C = K − 273.15
  • F → K: K = (F − 32) × 5/9 + 273.15
  • K → F: F = (K − 273.15) × 9/5 + 32

Example

Convert 25°C to Fahrenheit:
F = (25 × 9/5) + 32 = 77°F
